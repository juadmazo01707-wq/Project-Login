# Project Login

## Description
This project implements an interface of user modern and elegant for form of login.
Uses a design with gradient effects for create an experience visual professional.

## Project files

### `login.html`
- Main file HTML what contain the structure of form of login.
- Implements a design responsive and accessible.
- Includes validaitons HTML5 for email and camps required.

### `login.css`
- Style sheet that defines the entire visual appearance.
- Uses modern effects such as backdrop-filter and gradients.
- Complement design responsive.

## Characteristics technical

### Functionalities 
- Form of login with validation HTML5.
- Camps of email and password.
- Icons decorative.
- Effects of hover in buttons.
- Responsive design for mbile and desktop.

### Technologies used
- **HTML5**: Semantic stucture and native validations.
- **CSS3**: 
  - Flexbox for layout.
  - Gradients radial and line.
  - backdrop-filter for glass effects.
  - Transformations and transitions.
  - Box-shadow for profundity.

### Design characteristics 
- **Theme**: Dark with gradients purple-blue.
- **Effects**: Glassmorphism with blur of fond.
- **Main colors**:
  - Fond: Radial gradient (#3b003b a #0d0d24).
  - Card: Semi-transparent linear gradient.
  - Button: Pink-purple gradient (#F06292 a #171a4a).

### CSS Key Features
- **Layout Center**: Use flexbox to center veertically and horizontally.
- **Glassmorphism**: backdrop-filter: blur(20px) in the card.
- **Gradients**: Multiple grdients for create profundity.
- **Responsive**: Design adaptable to different screen sizes.

## Use instructions

### Installation
1. All download the files of project.
2. Ensure that `login.html` and `login.css` are in the same folder.
3. Not external dependencies required.

### Execution
1. Open `login.html` in any modern web browser.
2. The design will automatically adapt to your device.
3. Test the validation by entering data in the fields.

### Personalization
To customize the design:

1. **Colors**: Modify the color variables in `login.css`.
2. **Gradients**: Adjust the values in `background: linear-gradient()` and `background: radial-gradient()`.
3. **Dimensions**: Change `width` and `padding` in `.login-card`.
4. **Icons**: Replace emojis in HTML with custom icons.
